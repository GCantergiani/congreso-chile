Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install scrap_congreso


Or clone the repo:

    $ git clone https://github.com/Gcantergiani/scrap_congreso.git
    $ python setup.py install
